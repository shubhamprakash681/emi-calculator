## Installation & Setup

Clone the repository:

```bash
    git clone https://github.com/shubhamprakash681/emi-calculator.git
```

Navigate to the project directory:

```bash
    cd emi-calculator
```

Install dependencies:

```bash
  npm install
```

Run the development server:

```bash
  npm run dev
```

Open in browser:

```bash
  http://localhost:5173/
```
